# Keeper 📒

**Книга Учёта Доходов и Расходов** — современное TUI-приложение для ведения бизнеса в стиле neovim.

![Python](https://img.shields.io/badge/Python-3.10+-blue)
![Textual](https://img.shields.io/badge/Textual-0.52+-green)

## ✨ Особенности

- 🎨 **Современный TUI** — интерфейс в стиле LazyVim с цветовой схемой Dracula
- 📊 **Excel-интеграция** — все данные сохраняются в .xlsx файлы
- ⌨️ **Vim-style навигация** — управляйте с клавиатуры (j/k, h/l)
- 📁 **Разделение периодов** — создавайте новые файлы для каждого месяца/налогового периода
- 💰 **Автоматический подсчёт** — доходы, расходы и баланс в реальном времени

## 🚀 Быстрый старт

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
python keeper.py
```

## 📖 Использование

### Создание нового файла
Нажмите `n` и выберите место для сохранения Excel-файла.

### Добавление транзакции
1. Нажмите `a`
2. Заполните поля:
   - **Дата** — формат ГГГГ-ММ-ДД (по умолчанию сегодня)
   - **Название** — описание операции
   - **Сумма** — числовое значение
   - **Тип** — переключатель Доход (IN) / Расход (OUT)
3. Нажмите Enter или кнопку "Сохранить"

### Редактирование транзакции
1. Выберите строку навигацией `j/k` или `↑/↓`
2. Нажмите `e`
3. Измените поля и сохраните

### Удаление транзакции
1. Выберите строку
2. Нажмите `d`
3. Подтвердите удаление

### Разделение периода (Split)
Нажмите `s` для создания нового файла. Текущий файл остаётся без изменений — данные не копируются.

## ⌨️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `n` | Новый файл |
| `o` | Открыть файл |
| `a` | Добавить транзакцию |
| `e` | Редактировать |
| `d` | Удалить |
| `s` | Split (новый период) |
| `j` / `↓` | Вниз |
| `k` / `↑` | Вверх |
| `q` | Выход |
| `Enter` | Подтвердить (в модале) |
| `Esc` | Отмена (в модале) |
