## 📖 Использование

### Создание нового файла
Нажмите `n` и выберите место для сохранения Excel-файла.

### Добавление транзакции
1. Нажмите `a`
2. Заполните поля:
   - **Дата** — формат ГГГГ-ММ-ДД (по умолчанию сегодня)
   - **Название** — описание операции
   - **Сумма** — числовое значение
   - **Тип** — переключатель Доход (IN) / Расход (OUT)
3. Нажмите Enter или кнопку "Сохранить"

### Редактирование транзакции
1. Выберите строку навигацией `j/k` или `↑/↓`
2. Нажмите `e`
3. Измените поля и сохраните

### Удаление транзакции
1. Выберите строку
2. Нажмите `d`
3. Подтвердите удаление

### Разделение периода (Split)
Нажмите `s` для создания нового файла. Текущий файл остаётся без изменений — данные не копируются.

## ⌨️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `n` | Новый файл |
| `o` | Открыть файл |
| `a` | Добавить транзакцию |
| `e` | Редактировать |
| `d` | Удалить |
| `s` | Split (новый период) |
| `q` | Выход |
| `Enter` | Подтвердить (в модале) |
| `Esc` | Отмена (в модале) |

## 📁 Структура Excel-файла

```
Лист: "Транзакции"
| A (Дата) | B (Название) | C (Сумма) | D (Тип) |
|----------|--------------|-----------|---------|
| 2024-01-15 | Продажа товара | 15000.00 | IN |
| 2024-01-16 | Аренда офиса | 25000.00 | OUT |
```
